# GIT-TDD
# Project Name:
  
  Cats List

# Description:  

It is a Website for adopting and donating cats.

# Technologies:

- MEAN stack 

# Installation:

- fork and clone.
- redirect the command to the repo folder.  
- npm install.

# Run the Unit Test:

- npm run unit-test

# Run the Server:

- npm start
- make sure that you are running mongo DB in the Terminal:
  sudo mongod
